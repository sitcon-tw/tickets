## 開發

此系統使用 Astro 與 Fastify，基於 [Fastro](https://github.com/Edit-Mr/Fastro) 模板開發。

如果只有要跑前端／後端記得先 `cd` 到相應的資料夾。一起跑就在根目錄。

```bash
pnpm i
pnpm dev
```

## i18n

在每個元件獨立編輯，不會全部塞在同一個大 JSON 。看了你就懂了不多解釋。
